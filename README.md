Для запуска программы в main.py надо ввести следующее (в переменные):
1. Токен от Вашего сообщества вк
2. Ваш access_token
3. Создать у себя бд и ввести название бд в DB_NAME
4. Ввести пароль от бд в DB_PASSWORD
